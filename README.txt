PASOS RECOMENDABLES A SEGUIR A LA HORA DE VISUALIZAR EL PROYECTO:

   1. Tener XAMMP instalado. Inicia Apache y MySQL desde XAMPP-control.

   2. Mueve la carpeta /influweb y /example-influweb a /htdocs de XAMPP.
	/influweb es el proyecto.
	/influweb-example es un ejemplo estático de cómo podría quedar la web.

   3. Levantar la BD en phpMyAdmin con el script SQL -> /influweb/bd-script.sql

   4. Abrir proyecto en navegador -> localhost:8080/influweb

   5. Al principio verás todo muy vacío. Accede como administrador (admin@influweb.com / password: 111) y sigue los pasos.

   6. Imagina que eres el administrador de la web y ve personalizando las cosas (crea eventos, añade sponsors y productos).

   7. Una vez termines la personalización, accede de nuevo pero esta vez regístrate como usuario y observa los resultados.
